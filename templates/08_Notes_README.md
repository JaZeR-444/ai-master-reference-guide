## File Naming Convention
- `YYYY-MM-DD_Note_[Topic].md`
- `YYYY-MM-DD_Analysis_[Subject].md`
- `YYYY-MM-DD_Meeting_[Event].md`

Examples:
- `2025-12-15_Note_Pricing_Changes_Analysis.md`
- `2025-12-20_Analysis_Claude_vs_GPT4_for_Coding.md`

## Best Practices
- Date all notes
- Link to evidence in other folders
- Distinguish fact from opinion
- Update notes when new information contradicts old thinking
- Tag notes with topics for easy searching
