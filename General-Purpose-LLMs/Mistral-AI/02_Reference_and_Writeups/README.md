# 02_Reference_and_Writeups - Third-Party Analysis

## Purpose
This folder contains **third-party articles, blog posts, tutorials, and analysis** about the provider. These provide external perspectives and real-world usage insights.

## Content Types

### Articles & Blog Posts
- [ ] Technical deep-dives
- [ ] Feature announcements and reviews
- [ ] Integration tutorials
- [ ] Use case studies
- [ ] Industry analysis
- [ ] Comparison articles

### Source Quality Tiers
**Tier 1 (Highly Credible)**
- Technical blogs from recognized experts
- Peer-reviewed technical publications
- Official partner case studies

**Tier 2 (Credible)**
- Established tech publications (TechCrunch, VentureBeat, Ars Technica)
- Company engineering blogs
- Conference presentations

**Tier 3 (Anecdotal)**
- Individual blog posts
- Reddit/forum discussions
- Twitter threads
- YouTube reviews

## Organization Guidelines
- Create a markdown index: `_INDEX.md` listing all saved articles
- Save articles as markdown or PDF
- Include source URL and archive date in filename
- Tag articles by topic: `[Pricing]`, `[Tutorial]`, `[Comparison]`, etc.

## File Naming Convention
- `YYYY-MM-DD_Author_Title.md`
- `YYYY-MM-DD_[Tag]_Brief_Description.pdf`

Examples:
- `2025-12-15_Simon_Willison_Claude_Computer_Use_Review.md`
- `2025-11-20_[Comparison]_GPT4_vs_Claude_Coding.pdf`

## Critical Evaluation
When saving articles, note:
- **Bias**: Is the author affiliated with the provider?
- **Recency**: Is the information still current?
- **Evidence**: Are claims backed by data or anecdotal?
