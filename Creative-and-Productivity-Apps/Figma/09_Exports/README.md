# 09_Exports - Final Research Outputs

## Purpose
This folder contains **completed research reports, final summaries, and publication-ready analyses** that synthesize all research from the other folders.

## Content Types

### Research Reports
- [ ] Deep research analysis (following the research prompt template)
- [ ] Comparative studies
- [ ] Market analysis
- [ ] Strategic recommendations

### Summaries
- [ ] Executive summaries
- [ ] Quick reference guides
- [ ] Decision matrices
- [ ] Capability snapshots

### Shareable Outputs
- [ ] Blog post drafts
- [ ] Presentation slides
- [ ] Infographics
- [ ] Video scripts

## Quality Standards

All exported research should:
- ✓ Be comprehensive and evidence-based
- ✓ Cite sources from other folders
- ✓ Include confidence levels for claims
- ✓ Distinguish fact from inference
- ✓ Be dated and versioned
- ✓ Include an executive summary
- ✓ Follow the research prompt structure (if deep research)

## File Naming Convention
- `YYYY-MM-DD_[ProviderName]_Deep_Research_Analysis.md` (main output)
- `YYYY-MM-DD_[ProviderName]_Quick_Reference.md`
- `YYYY-MM-DD_[ProviderName]_Competitive_Analysis.md`

## Research Report Checklist

Before finalizing a report:
- [ ] All 11 required sections completed (per research prompt)
- [ ] Model catalog is complete and current
- [ ] Pricing information is up-to-date
- [ ] Sources are cited inline
- [ ] Evidence quality is labeled (primary/secondary/anecdotal)
- [ ] Confidence levels are stated
- [ ] Strategic assessment includes specific use cases
- [ ] Report follows consistent formatting
- [ ] Spell check and grammar check completed
- [ ] Version number and date in header

## Version Control
When updating a research report:
- Keep previous versions with date suffix: `..._Analysis_v1_2025-12-15.md`
- Note major changes in a changelog section
- Update "Last Reviewed" date

## Export Guidelines

### Executive Summary (Required)
Every export should start with:
```markdown
# [Title]

**Date**: YYYY-MM-DD
**Analyst**: [Your Name / AI Agent]
**Version**: X.X
**Status**: [Draft / Final / Updated]
**Confidence**: [Low / Medium / High]

## Executive Summary
[3-5 sentences capturing the entire analysis]

## Key Findings
- Finding 1
- Finding 2
- Finding 3

## Strategic Recommendation
[Use / Consider / Avoid] - [Brief justification]
```
